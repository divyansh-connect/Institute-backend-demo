<h1># Institute ERP Backend API (demo)</h1>
<p> A scalable backend system for an Institute ERP built using Node.js, Express.js, and MongoDB. It handles students, fees, library, study materials, announcements, and authentication. <p>

<h2> Features </h2>
<ul>
<li>JWT Authentication (Login System)</li>
<li>Role-Based Access Control (Admin / Staff)</li>
<li>Student Management</li>
<li>Fee & Payment Tracking</li>
<li>Library Management (Seat + Membership)</li>
<li>Study Material (PDF Upload & Access)</li>
<li>Announcement System</li>
<li>Document Upload (firebase)</li>
<li>Request Validation using Joi</li>
<li>MVC Architecture</li>
</ul>
<h2> Tech Stack </h2>
<ul>
<li>Node.js</li>
<li>Express.js</li>
<li>MongoDB + Mongoose</li>
<li>JWT (jsonwebtoken)</li>
<li>bcryptjs (Password Hashing)</li>
<li>Firebase (File Storage)</li>
<li>Multer (File Upload)</li>
<li>Joi (Validation)</li>
</ul>

<h2> # Authentication</h2>
JWT-based authentication system <br>
Token required for protected routes <br>
Role-based access (Admin / Staff) <br>
